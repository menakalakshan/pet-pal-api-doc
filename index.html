    Create User


    Request 
    POST api/breed/create
    Parameter Required Type Description
    user_id True Integer User’s given ID
    name False String Username
    dob False date Date of Birth
    nic False String User’s NIC
    address False String Address of the user
    email True String User’s email address
    password True String Login Password
    phone False String User’s phone number
    role False String Admin/ Use

    Response
    Parameter Type Description
    id String Object ID
    status String confirmed or not_confirmed
    is_active Bool Status of the account
    is_first_time Bool Membership status
    is_deleted Bool Dataset status
    name String Date of birth of the pet
    nic Img NIC of the user
    email Integer Registered email
    password Integer Password for the account
    is_deleted bool Showing deleted status

    Pet Create CRUD

    Create Pet

    Request 
    POST api/pets/create
    Parameter Required Type Description
    pet_type True String Type of the Pet
    pet_name True String Pet’s name
    pet_birthday True Integer Birth date of the pet
    pet_age False Integer Pet’s age
    pet_breed True String Unique ID for Pet Breed
    gender True String Male/ Female
    pet_weight True Integer Weight of the pet
    pet_image True Img Profile image of the pet
    user_id True Integer Unique ID for the pet
    neck_collar_id False Integer Unique ID From the Neck Collar

    Response
    Parameter Type Description
    pet_type String Saved Type of the Pet
    pet_name String Saved Pet’s name
    pet_birthday Integer Saved Birth date of the pet
    pet_age Integer Saved Pet’s age
    pet_breed String Saved Unique ID for Pet Breed
    gender String Saved Male/ Female
    pet_weight Integer Saved Weight of the pet
    pet_image Img Saved Profile image of the pet
    user_id Integer Saved Unique ID for the pet
    neck_collar_id Integer Saved Unique ID From the Neck Collar
